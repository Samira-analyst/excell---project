# Coffee Orders Dashboard

## Overview

This project is an Excel dashboard created to analyze coffee orders, sales performance, and customer purchasing behavior.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

## Features

* Sales performance analysis
* Product category insights
* Customer order analysis
* Interactive dashboard filters
